Applications: Calculate Payment Date#


  1. Import and if necessary install modules

  2. Create a dictionary of non-settlement days for 2026 and 2028

  3. Use while loop to check if dates are in non-settlement dictionary or a weekend

from datetime import datetime, date, timedelta
try:
    from dateutil.easter import easter
except:
    !pip -q  install python-dateutil
    from datetutil.easter import easter

non_settlement_2026_2028_us=holidays.US(years=[2026,2028])
non_settlement_2026_2028_us[easter(2026)-timedelta(days=2)]='Good Friday'
non_settlement_2026_2028_us[easter(2028)-timedelta(days=2)]='Good Friday'

july_4_2026=date(2026,7,4)
july_4_2028=date(2028,7,4)
for july_4 in (july_4_2026,july_4_2028):
  adjusted_date=july_4
  while adjusted_date.weekday()>4 or adjusted_date in non_settlement_2026_2028_us:
    adjusted_date+=timedelta(days=1)
    if adjusted_date.year not in non_settlement_2026_2028_us:
      non_settlement_2026_2028_us.update(years=date.year)
  display(adjusted_date)

datetime.date(2026, 7, 6)

datetime.date(2028, 7, 5)

Chapter Three Exercise#

  1. Create a list of date objects.

  2. Create a list of coupon amounts

  3. Create a dictionary with keys equal to maturity and values coupon amount

  4. Make settlement equal to the date object for January 21\(^{st}\) 2025.

  5. Iterate through the dictionary with items() method of dictionaries.

  6. Make the keys, values, and settlement the arguments of bond_pay_data() and display the results for each iteration.

If you are unfamiliar with the items() method of dictionaries take a look at the background material.

from basic_concepts_fixed_income import (bond_pay_data)
settlement=date(2025,1,21)
maturity_dates=[date(2028,10,9),
            date(2028,12,31),
            date(2030,11,28),
            date(2027,10,10),
            date(2029,2,18)]
coupon_amounts=[1.5,5,3.5,2.5,4]
pay_data_dict={maturity:amount for maturity,amount in zip(maturity_dates,coupon_amounts)}
for key,value in pay_data_dict.items():
  display(bond_pay_data(key,value,settlement=settlement))
